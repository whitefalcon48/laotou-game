# 抓捕王二勇

HTML/CSS/JavaScriptだけで動く、低解像度ドット絵風の2D横スクロール風ミニゲームです。

## 遊び方

1. `index.html` をブラウザで開きます。
2. `开始` ボタン、またはタイトル画面で Space キーを押してスタートします。
3. `↑` `↓` `←` `→` または `W` `A` `S` `D` で老头を移動します。
4. Space キーで茶叶蛋を投げます。
   スマホでは画面下の触控ボタンで移動・投茶叶蛋・暂停ができます。
5. 小さい王二勇を15体倒すと「真・王二勇 発見！」が出て、ボスが登場します。
6. ボスを倒すとクリアです。

## 仕様

- HPは3です。
- 茶叶蛋は最大15個で、少しずつ補充されます。
- 敵に接触するか、敵を逃すとHPが減ります。
- 小さい王二勇やボスの攻撃に当たるとHPが減ります。
- 敵に茶叶蛋が当たると煙になって消えます。
- UIは左上にHPと茶叶蛋数、右上に `已抓捕 x/15` を表示します。

## 開発メモ

- 外部ライブラリは使っていません。
- ゲーム画面は `canvas` に描画しています。
- `assets/` にはゲームで使用する画像とBGMを入れています。
- `player.png`、`enemy.png`、`boss.png`、`egg.png` は透過PNGとして読み込み、固定の切り抜き範囲で表示しています。
- BGMは `bgm_main.mp3` と `bgm_boss.mp3` を使用しています。
- `game.js` には、非エンジニアにも追いやすいようにコメントを入れています。
- 追加の画像や音声を使う場合は `assets/` に入れてください。
